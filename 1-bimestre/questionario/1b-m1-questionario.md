Exercício 1
Algoritmo é um conjunto de passos organizados para resolver um problema.

A lógica de programação é o jeito de pensar na solução, e o algoritmo é essa solução escrita passo a passo.

Exemplo do dia a dia: fazer café Primeiro coloca água, depois o pó no filtro, esquenta a água, passa no filtro e depois serve.

Exercício 2
Algoritmo em linguagem natural:

Ler três números, comparar eles e mostrar qual é o maior.

Em Portugol:

programa
{
    funcao inicio()
    {
        inteiro a, b, c
        inteiro maior

        leia(a)
        leia(b)
        leia(c)

        maior = a

        se (b > maior) entao
        {
            maior = b
        }

        se (c > maior) entao
        {
            maior = c
        }

        escreva("O maior numero é: ", maior)
    }
}
